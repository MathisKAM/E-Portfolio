## 🚀 Technologies utilisées

- **JavaScript Vanilla** pur (pas de frameworks)
- **CSS3** avec animations, transforms, gradients
- **Font Awesome 6.4.0** pour toutes les icônes
- **Google Fonts** : Inter (300-800) & Poppins (300-800)
- **Intersection Observer API** pour les animations au scroll

## 📱 Responsive

- ✅ Desktop (1400px+)
- ✅ Tablette (768px - 1024px)
- ✅ Mobile (< 768px)

## ⚡ Performances

- Aucune bibliothèque externe lourde
- CSS optimisé avec variables
- Animations GPU-accelerated (transform, opacity)
- Lazy loading pour les sections

## 🎯 Résultat

Portfolio moderne, fluide, interactif avec :
- **Animations continues** sur éléments clés
- **Scroll animations bidirectionnelles**
- **Effets 3D** subtils mais impactants
- **Design cohérent** avec palette harmonieuse
- **Navigation intuitive** avec feedback visuel
- **Responsive** sur tous devices
