🐛 Debug des animations d'activités

Outils de debug :

Cliquez sur les boutons ci-dessus pour lancer les tests...

Instructions :

  1. Ouvrez le portfolio dans un nouvel onglet
  2. Ouvrez la console développeur (F12)
  3. Revenez ici et cliquez sur "Vérifier les éléments .activity"
  4. Si aucun élément n'est trouvé, le problème est dans le rendu
  5. Si les éléments sont trouvés mais sans animations, le problème est dans les événements
