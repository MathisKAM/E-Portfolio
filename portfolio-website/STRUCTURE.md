# Structure des fichiers

## Fichiers principaux

- `index.html` - Page principale du portfolio (à la racine pour GitHub Pages)
- `css/main.js` - Script principal avec toute la logique du portfolio
- `css/main.css` - Styles principaux du portfolio

## Organisation des assets

- `assets/images/` - Toutes les images (photos, logos, icônes)
- `assets/documents/` - CV et documents PDF
- `assets/fonts/` - Polices personnalisées (si utilisées)

## Scripts JavaScript

- `js/main.js` - Script principal avec composants et logique
- `js/modern-effects.js` - Effets visuels et interactions
- `js/animations.js` - Animations supplémentaires
- `js/i18n.js` - Gestion des langues (FR/EN)

## Styles CSS

- `css/main.css` - Styles de base et layout
- `css/animations.css` - Animations CSS personnalisées
- `css/modern-effects.css` - Effets visuels avancés

## Chemins importants

Tous les chemins sont relatifs à la racine du projet :
- Images : `assets/images/filename.ext`
- Documents : `assets/documents/filename.pdf`
- CSS : `css/filename.css`
- JS : `js/filename.js`

## Déploiement GitHub Pages

Le fichier `index.html` est à la racine, donc le site sera automatiquement accessible via GitHub Pages à l'adresse :
`https://username.github.io/repository-name/`
